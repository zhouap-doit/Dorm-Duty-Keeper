# UniApp + Vue3 移动端登录页面

这是一个基于 UniApp + Vue3 的移动端登录页面项目，提供了完整的登录功能和美观的UI设计。

## 🚀 项目特性

- **现代化设计**：采用渐变背景、圆角设计、阴影效果
- **响应式布局**：适配各种移动设备屏幕
- **完整功能**：账号密码登录、表单验证、错误提示
- **交互体验**：触摸反馈、加载状态、成功提示
- **代码规范**：使用 Vue3 Composition API、模块化设计

## 📱 页面结构

```
src/
├── pages/
│   ├── login/
│   │   └── login.vue          # 登录页面
│   ├── register/
│   │   └── register.vue       # 注册页面
│   └── index/
│       └── index.vue          # 首页
├── api/
│   ├── auth.js               # 认证相关API
│   └── register.js           # 注册相关API
├── utils/
│   └── path.js               # 路径工具
├── pages.json                # 页面路由配置
└── uni.scss                  # 全局样式变量
```

## 🎨 设计特色

### 视觉设计
- **渐变背景**：淡蓝到淡紫渐变营造轻盈现代感
- **圆角设计**：30-50rpx圆角，现代友好
- **阴影效果**：多层次阴影增强立体感
- **配色方案**：主色调#a6c0fe到#c2a8fd的渐变
- **毛玻璃效果**：backdrop-filter实现磨砂玻璃效果

### 交互体验
- **输入框聚焦**：边框变色+阴影效果
- **按钮触摸**：按下状态变化
- **表单验证**：实时错误提示
- **加载状态**：登录过程反馈

## 🔧 使用方法

### 1. 启动项目
```bash
# 安装依赖
npm install

# 运行到微信小程序
npm run dev:mp-weixin
```

### 2. 测试登录
- 用户名：`admin`
- 密码：`123456`

### 3. 自定义配置
- 修改 `src/api/auth.js` 中的API地址
- 调整 `src/uni.scss` 中的样式变量
- 更新 `src/pages.json` 中的页面配置
