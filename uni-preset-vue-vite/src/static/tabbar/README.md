# TabBar 图标说明

此目录需要包含以下图标文件（建议尺寸：81x81px）：

- `home.png` - 首页未选中图标
- `home-active.png` - 首页选中图标
- `duty.png` - 值日表未选中图标
- `duty-active.png` - 值日表选中图标
- `mine.png` - 我的未选中图标
- `mine-active.png` - 我的选中图标

图标要求：
- 格式：PNG
- 尺寸：建议 81x81px（uni-app tabBar 标准尺寸）
- 颜色：未选中图标使用灰色，选中图标使用绿色（#4CAF50）

如果图标文件不存在，页面将使用 emoji 作为后备显示。
